# 介绍

使用本python脚本，可通过苹果官方rss接口获取指定应用在appstore的最新500条评论

# 使用方式：

1.安装python 3环境，不会的请百度

2. pip安装模块： XlsxWriter，用来生成excle文件 

```shell
pip install XlsxWrite
```

# 注意事项：

接口限制。最多获取500条；不足500条的话，有多少获取多少。

# 关于如何获取ID号

百度搜索：appstore+app名，点第一个，进入网站，看地址栏。
